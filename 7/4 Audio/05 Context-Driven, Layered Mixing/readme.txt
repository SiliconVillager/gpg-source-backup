Application
-----------

The included sample program demonstrates context-driven, layer mixing.

Install and uninstall mixing snapshots by clicking the "Install" 
checkboxes.

The rightmost mixing interface visually shows the volumes levels of 
the central mix.  Turn up your speakers to hear the changes.

To edit a snapshot, use the "Edit" checkboxes and adjust 
volume settings using the editing mixing interface.

Please refer to the code and the article for more details.
