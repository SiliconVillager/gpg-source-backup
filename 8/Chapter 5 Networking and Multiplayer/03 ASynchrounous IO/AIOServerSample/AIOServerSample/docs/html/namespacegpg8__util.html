void gpg8_util::sleep ( double  seconds  ) 

Interface to a reasonably high-resolution sleep() function on the underlying platform.

Uses SleepEx() to yeild for event notifications, and then Sleep() to get an uninterrupted sleep for the specified duration.

This function should also enable the OS to dispatch any notifications that require a cooperative yield from the application.
