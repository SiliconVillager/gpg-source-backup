Detailed Description

The GameInstance class models one game session, which consists mainly of a collection of connections from the players and a shared gamestate.

In this model, the game state is just a struct padded out with an array to simulate whatever amount of data is specified by the socket's MAX_WRITE_SIZE. Similarly, this class models input from the players as another padded struct sized to match the MAX_READ_SIZE from the socket.
