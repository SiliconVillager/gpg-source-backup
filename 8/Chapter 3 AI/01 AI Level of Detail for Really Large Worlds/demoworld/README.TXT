
Prerequisities
--------------
IVE Demo World requires IVE-1.1 and JDK 1.6 or higher. The compilation process is 
based on ant 1.7


Running Demo World in IVE
-----------
Run ive either by executing 
java -jar ive_core.jar
or by clicking twice on the jar file icon in windows.

Then, in IVE GUI menu, select the Load World item and select the 
demoworld/dist/DemoWorld.xml file.  Setup holdbacks, open important locations and 
press the play button.


Compilation
-----------
To compile DemoWorld, switch to the project root directory containing the build.xml 
file and run:

ant
