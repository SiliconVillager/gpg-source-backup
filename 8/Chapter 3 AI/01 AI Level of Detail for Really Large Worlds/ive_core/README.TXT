
Prerequisities
--------------
IVE requires JDK 1.6 or higher. The compilation process is based on ant 1.7


Running IVE
-----------
IVE is distributed as compiled jar file which can be run by standard Java means.
To run the IVE, copy the directory ive_core to your harddrive and switch to the 
dist directory containing the ive_core.jar file. Run

java -jar ive_core.jar

On Windows, you can alternatively run the jar file by clicking on its icon twice 
(if associations in your system are set correctly by Java installation)


Compilation
-----------
To compile IVE, switch to the project root directory containing the build.xml 
file and run:

ant
